## Setup Tutorial

Step 1: Clone source
```
git clone https://github.com/nhockool1002/wordpress-docker-base.git
```

Step 2: Redirect to project
```
cd wordpress-docker-base
```

Step 3: Run project (require: docker installed)
```
docker-compose up -d
```
